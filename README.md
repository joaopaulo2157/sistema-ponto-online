# Sistema de Ponto Online Simplificado

Este é um projeto simples em PHP ve MySQL para registro de ponto de colaboradores

## Como Instalar
1. Clone este repositório.
2. Importe o arquivo SQL localizado em `/sql/setup.sql` no seu PHPMyAdmin.
3. Ajuste as ceredenciais cde bando de dados em `includes/db.php`.
4. Acesse via navegador no seu servidor local (XAMPP/WAMP).

## Tecnologias

- PHP
- MySQL
- Water.css(Interface)
